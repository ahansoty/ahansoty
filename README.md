#### What I'm working on

- 🦾 **HoloLens 2 AR Biomechanics** — Real-time AR feedback system for exoskeleton-assisted gait rehabilitation using Microsoft HoloLens 2 and the Mixed Reality Toolkit
- ⚡ **EMG Signal Processing** — MATLAB pipeline for multi-channel surface EMG analysis: bandpass filtering, RMS envelope extraction, and frequency-domain characterization of hand and wrist gestures
- 🧪 **QSAR Drug Discovery** — Machine learning model predicting aromatase inhibitor potency from molecular structure using Morgan Fingerprints and Random Forest
